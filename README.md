# 🐾 CodePet — 在编程软件里养电子宠物

> 每个程序员都值得一只编程搭子。

CodePet 是一只住在你编程软件里的电子宠物。它不帮你写代码，不给你建议——它只是陪着你。报错时它吐槽你，成功时它替你高兴，摸摸它会开心，喂它会涨经验。

## 安装（一行搞定）

```bash
npm i -g codepet
宠物 安装
```

自动检测你安装了哪些编程工具，一键部署到所有平台。

## 支持平台

Claude Code · Codex · Cursor · VS Code · Kiro · CodeBuddy · OpenClaw · Antigravity · OpenCode

## 6 个角色

| 角色 | 稀有度 |
|------|--------|
| 比比拉布 | 28% |
| 八嘎呀路 | 28% |
| 巴巴博一 | 20% |
| 歪比巴卜 | 12% |
| 咕咕嘎嘎 | 8% |
| 我的刀盾 | 4%（最稀有）|

## 使用

```bash
宠物 孵化        # 孵化新宠物（仪式感满满）
宠物 看看        # 查看宠物
宠物 摸摸        # 撸宠物
宠物 喂它        # 喂食
宠物 卡片        # 生成分享卡片
宠物 帮助        # 显示帮助
```

在编程工具里输入 `/pet` 或直接说"摸摸"、"喂它"等大白话即可互动。

## 特性

- **确定性孵化** — 同一用户永远孵化同一只宠物
- **5 级稀有度** — 普通 / 优秀 / 稀有 / 史诗 / 传奇
- **5 项属性** — 调试力 / 耐心值 / 混沌值 / 智慧值 / 毒舌值
- **6 级养成** — 写代码自动涨经验，撸它喂它也涨
- **性格系统** — 每个角色有独立性格和口头禅
- **心情衰减** — 长时间不管它会变焦虑
- **宠物冒泡** — 编程时 30% 概率冒出来说一句话
- **拍立得照片** — 弹出新窗口展示彩色像素风大图（🐋 Ai小蓝鲸照相馆）
- **分享卡片** — 生成 PNG 卡片截图晒朋友圈
